# Scripts

Thư mục chứa script hỗ trợ dev, build và deploy.

## Danh sách

- `dev-start.sh`: khởi động service dev nền bằng Docker trên Linux/macOS.
- `dev-start.bat`: khởi động service dev nền bằng Docker trên Windows.
- `build-client.sh`: build React client.
- `build-server.sh`: publish .NET API.
- `deploy-ubuntu.sh`: khung deploy Ubuntu.
- `../run-windows.bat`: file chạy nhanh trên Windows từ thư mục gốc, double-click để mở client/server.

## Ghi chú

Script hiện là khung ban đầu. Cần cập nhật domain, path, user server và biến môi trường thật trước khi dùng production.
