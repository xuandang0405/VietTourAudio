# Logo Assets

Thư mục lưu logo dùng chung cho VietTourAudio.

- `placeholder-logo.png`: placeholder cũ ở root assets.
- Frontend chính dùng `client/src/assets/logo/logo.png` và `client/src/assets/logo/logo-text.png`.

Khi có bộ nhận diện chính thức, thay file logo thật nhưng giữ đường dẫn import trong frontend.
