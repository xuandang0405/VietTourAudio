# Database Design

Tài liệu database mới nằm ở `docs/DATABASE_GUIDE.md`.

Database bắt buộc là `viettuoraudio`. Các bảng chính gồm `users`, `stalls`, `pois`, `poi_contents`, `media_files`, `qr_codes`, `qr_scan_events`, `visit_events`, `play_history`, `payments`, `cash_reports`, `commissions`, `admin_logs`, `app_settings`.
