# Client VietTourAudio

Frontend là ReactJS PWA dùng Vite.

## Chạy local

```bash
npm install
npm run dev
```

## Logo

- `src/assets/logo/logo.png`: favicon, app icon, navbar icon, loading screen.
- `src/assets/logo/logo-text.png`: navbar, landing page, footer.

## Cấu hình API

Tạo `client/.env`:

```env
VITE_API_BASE_URL=http://localhost:5000/api
VITE_DEFAULT_LANGUAGE=vi
```

## Build

```bash
npm run build
```
